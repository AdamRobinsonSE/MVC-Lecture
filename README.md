# To get the app working

1. npm init
2. npm install
3. You will need to add a .env file that contains PORT and DB_STING
```js
PORT = 8000

DB_STRING = mongodb+srv://<name>:<password>@cluster0.srqu63o.mongodb.net/?retryWrites=true&w=majority
```
4. npm start
